## INFORMATION ##

Avatar Gallery Manager (1.0) for MyBB 1.6
Created by: Starpaul20
Copyright: ©2012
License: GPL

Allows administrators to manage the avatar gallery from the Admin CP, allowing them to upload, rename, move or delete avatars or galleries and update any users using them.

This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Activate"
4. Enjoy!


## UPDATING ##

Any beta version should be reinstalled.
